Mobile application for Felgen Wie Neu Wheel-Rim-Organizer
==============

## Using this project

To start the project run commands below:

```bash
$ bower update
```

```bash
$ npm install
```

Then to run in browser: 

```bash
$ ionic serve
```
To run on android device:

```bash
$ ionic run android
```

To run on ios device:

```bash
$ ionic build ios
```

Then run build in X-code.
